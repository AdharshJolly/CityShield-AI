# Research Guardrails

## Allowed

- Better Augmentation
- Better Datasets
- Better Evaluation
- Better Analytics
- Model Optimization
- Quantization
- Inference Improvements

## Restricted

- Foundation Model Training
- Reinforcement Learning
- Federated Learning
- New Architecture Research
- Multi-modal Foundation Models

## Principle

This is a 2-week hackathon.

Prefer execution over novelty.

Novelty must improve judging outcomes and remain implementable.
