# CityShield AI - Project Constitution

## Mission
Build a Public Safety Intelligence Platform capable of detecting, analyzing, and prioritizing urban hazards from CCTV feeds.

---

## Non-Negotiable Rules

1. Do not redesign the system architecture without team approval.
2. Follow the PRD, TAD, SDD, and ML Specification documents.
3. Maintain compatibility with the Unified Hazard Detector architecture.
4. Maintain compatibility with the Accident Detection Engine.
5. Avoid introducing new models unless a measurable benefit is demonstrated.
6. Prioritize completion over experimentation.
7. Keep solutions production-oriented.
8. All code must be documented.
9. All changes must be traceable.
10. Optimize for hackathon delivery within 2 weeks.
